# Modern Freelancer Portfolio Template

A stylish and fully responsive React portfolio template for developers, freelancers, and creative professionals.

## ✨ Features
- Modern, clean UI
- Ready-to-customize: change images, colors, text easily
- “Download CV” button (replaceable CV)
- Featured projects grid
- Skills, about, and contact form
- Font Awesome icons (via react-icons)
- Fast, SEO-friendly, mobile-ready
